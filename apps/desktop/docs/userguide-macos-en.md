# BioMolExplorer — User Guide (macOS)

> **macOS support is not yet available.**
>
> A native macOS installer is currently under development. This page will be updated when a stable release is ready.

---

## What to Expect

When macOS support is released, installation will follow these steps:

1. Download the `BioMolExplorer.dmg` file.
2. Open the `.dmg` and drag the BioMolExplorer icon to the **Applications** folder.
3. Open **BioMolExplorer** from the Applications folder or via Spotlight.

On the **first launch**, the application will automatically set up the scientific environment (Miniconda, Node.js, RDKit, OpenBabel, Vina, PyMOL, Flask) — no manual configuration required.

---

## In the Meantime

If you need to use BioMolExplorer now, consider the following alternative:

- **Linux (recommended):** Use a native Linux installation or a virtual machine with Ubuntu 20.04+. Install the package with `sudo apt install ./biomolexplorer_amd64.deb` — see the Linux user guide for details.
