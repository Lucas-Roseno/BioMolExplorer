# BioMolExplorer — Guia do Usuário (macOS)

> **O suporte a macOS ainda não está disponível.**
>
> Um instalador nativo para macOS está em desenvolvimento. Esta página será atualizada quando uma versão estável for lançada.

---

## O Que Esperar

Quando o suporte a macOS for lançado, a instalação seguirá estas etapas:

1. Baixe o arquivo `BioMolExplorer.dmg`.
2. Abra o `.dmg` e arraste o ícone do BioMolExplorer para a pasta **Aplicativos**.
3. Abra o **BioMolExplorer** pela pasta Aplicativos ou via Spotlight.

Na **primeira abertura**, o aplicativo irá configurar automaticamente o ambiente científico (Miniconda, Node.js, RDKit, OpenBabel, Vina, PyMOL, Flask) — nenhuma configuração manual é necessária.

---

## Enquanto Isso

Se você precisar usar o BioMolExplorer agora, considere a seguinte alternativa:

- **Linux (recomendado):** Use uma instalação nativa de Linux ou uma máquina virtual com Ubuntu 20.04+. Instale o pacote com `sudo apt install ./biomolexplorer_amd64.deb` — veja o guia do usuário para Linux para mais detalhes.
